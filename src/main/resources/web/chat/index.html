    当前助手采样参数

    限制单次回复的最大 token 数量，留空时使用服务端默认值

    控制生成文本的随机性。值越高输出越随机，值越低输出越集中确定

    核采样阈值，只在累计概率达到阈值的候选 token 中采样

    每一步只保留概率最高的前 K 个候选 token 参与采样

    过滤掉概率低于阈值的 token，降低低概率噪声输出

    对已经出现过的 token 施加惩罚，鼓励模型引入新内容

    抑制近期重复 token，减少模型连续复述相同内容

    根据 token 出现频率增加惩罚，降低模板化和高频重复表达

    每行填写一个停止序列，命中后模型会停止继续生成
